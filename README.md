# Tabs to text

This Chrome extension allows you to quickly copy information from all open tabs to your clipboard.

Uses manifest v3.

## Default Copy format

```
- (title) | (url)
```

## Customizable format 

Define your own custom format from the popup.

Use `${title}` and `${url}` as placeholders in the custom format.

```
- [${title}](${url})
```
